---
title: remark/rehype
---

Pass `rehype/mark` plugins inside a `mdxBundlerOptions` object.

```ts {7,10} showLineNumbers
import { bundle } from 'mdx-tug';

// ...

return bundle({
  // other options
  mdxBundlerOptions: {
    mdxOptions: {
      jsxImportSource: 'preact',
      remarkPlugins: [
        /* add or change remark plugins*/
      ],
      rehypePlugins: [
        /* add or change rehype plugins*/
      ],
    },
  },
});
```
